# StatVision - Strategic Feature Roadmap

Please mark the items you want to prioritize by replacing `[ ]` with `[x]`. Once you've made your selections, save the file and let me know.

### 🏀 Rich Features (The "Amateur Experience")
- [ ] **The "Ghost Scoreboard" Overlay:** Automatically overlay a digital scoreboard and game clock onto the raw video based on AI-detected points and period changes.
- [*] **Automatic "Mixtape" / Highlight Generator:** A one-click button that cuts and stitches "Made Shots," "Blocks," and "Steals" into a 30-60 second social-media-ready clip.
- [*] **"Visual Roster" Pairing (No Jerseys):** Allow users to tag players by visual markers (shoe color, height, hair style) instead of just jersey numbers for informal pickup games.
- [*] **The "Park Heatmap" (Community Stats):** Aggregate data from multiple games at the same physical court (e.g., "Main Street YMCA") to show common "hot spots" and local leaders.
- [*] **"Hustle & Vibe" Metrics:** Track non-traditional stats like "Screen Assists," "Contest Rate," and "Boxing Out" that are usually invisible in amateur play.
- [*] **Voice-Activated "Live Tagging":** A simple mobile button or voice command ("Score Blue!") during the game to help the AI anchor its analysis later.

### ⚙️ Technical Proposals (The "Under-the-Hood" Edge)
- [*] **Edge-Side Pre-processing (Mobile-First):** Implement client-side compression or frame extraction to reduce upload sizes and save mobile data/bandwidth at the park.
- [*] **Multi-Angle Syncing (Crowdsourced):** Allow multiple phone recordings of the same game to be uploaded and synced via audio/whistle detection for 360-degree AI accuracy.
- [*] **"Halftime Report" Latency (Gemini 3 Flash):** Optimize the pipeline for speed so players can upload the first half and get a strategic summary before the second half starts.
- [*] **Vector-Based Spatial Search:** Implement "Natural Language Search" (e.g., *"Show me all the times we lost the ball on a double team"*) using vector embeddings.
- [*] **Pose Estimation for Form Coaching:** Add a specialized worker to analyze shooting mechanics (elbow flare, jump height, release point) for individual player development.
- [*] **Automated "Referee" Mode:** Use AI to flag potential violations (Travels, Double Dribbles, Fouls) to help settle arguments during high-stakes pickup games.

### 🌱 "Stalactite" Growth Tracking (Long-term)
- [*] **The "Evolution" Dashboard:** A long-term visualization of a player's "mineral accumulation"—tracking 1% improvements in specific skills (e.g., left-hand finishes) over a year.
- [*] **Player "Archetype" Discovery:** Automatically classify players into archetypes (e.g., "3-and-D Wing," "Glass Cleaner," "Floor General") based on their playstyle data.
- [*] **"Park Legends" Leaderboard:** A local or regional ranking system based on an "ELO" or "Skill Rating" derived from AI-analyzed game performance.
