# 🧪 StatVision Sandbox

This directory is used for temporary scripts, debug logs, experimental code, and local test configurations.

## 🔒 Security Policy
- All contents of this folder are **GIT IGNORED** by default (except for this README and `.gitkeep`).
- **Never** move production secrets or sensitive user data into tracked files.
- Use this space for one-off automation scripts (e.g., TBP browsing, API seeding) that should not clutter the main codebase.

## 🛠 Usage
- Put your `.env.test-user` or other local environment overrides here.
- Experimental `ts-node` scripts can be executed from this directory without affecting the build system.
